After successful firmware flashing, power on the device. The robot will enter network distribution prompt mode.
This firmware adopts the Xiaozhi chip solution, so its network pairing operation is exactly the same as the standard Xiaozhi ESP solution.
Search for Wi-Fi signals starting with xiaozhi xxx on your mobile phone. Connect to this hotspot, and the Wi-Fi configuration page will pop up automatically. Select your home Wi-Fi name and enter the password to finish network pairing.
Open your mobile browser or WeChat, and visit the URL:
https://wowing.love/
You can bookmark the page in WeChat for quick access next time, or add it to your mobile desktop via browser to use it like a standalone APP.
New users need to register for the first login. Select the correct country area code, enter your valid phone number, and set a 6-digit login password.
After logging in successfully, you can enter and use the robot intelligent control console.
